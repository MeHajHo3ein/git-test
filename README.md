## Installation

1. Clone this repository.

```bash
git clone https://github.com/mdpe-ir/md_financial
```

2. Naviagte to the project directory.

```bash
cd md_financial
```

3. Install dependencies.

```bash
flutter pub get
```

4. Run the app.

```bash
flutter run
```
